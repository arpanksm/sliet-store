# ğŸš€ Getting Started Locally

Follow these steps to run the project on your local machine.

1. Clone the repository

git clone [https://github.com/arpanksm/sliet-store.git](https://github.com/arpanksm/sliet-store.git)
cd sliet-store


2. Install Dependencies

npm install


3. Firebase Configuration

Create a project at Firebase Console.

Enable Authentication (Email/Password).

Enable Firestore Database.

Create a file named .env in the root directory.

Add your Firebase config keys (refer to src/App.jsx to see which keys are needed, or paste your config object).

4. Run the App

npm run dev


The app will start at http://localhost:5173.
